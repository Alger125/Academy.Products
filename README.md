# Academy.Products
# Épica: Gestión de Catálogo y Productos

## Descripción
El sistema debe permitir a los clientes visualizar, buscar y filtrar productos, 
mientras que los administradores gestionan el inventario.

## Historias de Usuario Relacionadas
- [ ] Como cliente, quiero buscar productos por nombre o categoría, para encontrarlos más rápido.
- [ ] Como cliente, quiero filtrar productos por precio, para ajustar la búsqueda a mi presupuesto.
- [ ] Como administrador, quiero agregar/editar/eliminar productos, para mantener actualizado el inventario.
- [ ] Como cliente, quiero visualizar los productos con imagen, descripción y precio, para decidir mejor mi compra.

## Alcance
- Implementación de API REST con C# y .NET.
- Uso de patrón CQRS para queries y commands.
- Manejo de errores y códigos HTTP estandarizados.

## Flujo final
- Desarrollas en feature/... → PR hacia develop.
- De develop → PR hacia QA.
- De QA → PR hacia prd.
- De prd → PR hacia main (producción estable).
